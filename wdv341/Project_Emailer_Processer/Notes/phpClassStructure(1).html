General structure of a Class in PHP

<?php

class className

//Comments describing the class, its purpose, special use cases, change history, etc.

Properties - This is a list of properties or variables defined within the class and stored by the class.

Constructor Functions - If needed start with the 'empty' constructor function.

Setter Functions - Functions used to set a value into a property. In the same order as the properties.

Getter Functions - Functions used to get a value from a property. In the same order as the properties.

Processing functions - Functions that will do something with various pieces of data for a variety of purposes.

Usually listed in alphabetical order.
