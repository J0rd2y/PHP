OOP Function - Accessors and Mutators (Getters and Setters)

Functions or Methods are critical when using OOP PHP. One of the first uses is to create methods to set the value of variables stored in a class or object. These are called Mutator Methods or "Setters" since they set or change the value in the variable. Accessor methods or Getters are used to pull the value of the variable into the program. They do not change anything in the class or object their job is to find the value and make it available outside the object.

Mutator Method / Set Method / Setters This type of method is defined within a Class. It is used to set the value of a property/variable within the object. These methods can be very simple or they can be quite complex requiring extensive programming in order to find and validate the value before being stored in the property. These methods almost always required an input parameter with the value to be stored.

Accessor Method / Get Method / Getters This type of method is defined within a Class. It is used to retrieve the value from a property in the object. These methods can be as simple as returning the variable value or they can include extensive processing in order to prepare the value for use outside the object. These methods often include a 'return' to handle the desired value or an output process that produces the desired value.

Example:

Class Product {

$productName; //property holding the name of the product

 

function setProductName($inProductName) {

$productName = $inProductName; //simple assigment through the Set method

}

function getProductName( ) {

return $productName; //sends the value stored in this object/variable out to the program

}

}//end Product Class
